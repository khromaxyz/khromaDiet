# PromptForge — Boost Library v4

Blocos de pressão de qualidade. Cada nível é um PARÁGRAFO, não uma lista.
A diferença entre níveis é intensidade da linguagem, não volume.

Princípio: a skill html-single-file já cobre anti-preguiça, continuação,
responsividade, semântica, e organização. O boost foca EXCLUSIVAMENTE em
elevar a ambição criativa e a qualidade visual.

---

## Boost 0 — Nenhum

Sem injeção. O prompt confia inteiramente na skill html-single-file.

---

## Boost 1 — Sutil

Adicionar ao final do prompt (1-2 frases):

```
Capricha no design. Quero abrir o arquivo e sentir que não parece genérico.
```

---

## Boost 2 — Presente (default)

Adicionar ao final do prompt (1 parágrafo):

```
Quero um resultado nível portfólio premium — algo que eu abriria e pensaria "isso é bom de verdade". Cada sombra com camadas, cada hover com personalidade, cada animação com timing pensado. Não jogue seguro com o design. Me surpreenda.
```

---

## Boost 3 — Forte

Adicionar ao final do prompt (2 parágrafos):

```
Este não é um exercício ou protótipo. É a peça central do meu projeto e precisa estar no nível dos melhores sites que eu já vi. Pense Awwwards, Linear, Stripe — esse patamar de polish. Cada pixel com intenção. Profundidade tridimensional real nas superfícies. Animações que parecem coreografadas, não aplicadas. Tipografia que tem presença.

Não jogue seguro. Faça algo que me faça parar e olhar. O tipo de interface que quando alguém vê, quer saber como foi feita.
```

---

## Boost 4 — Intenso

Adicionar ao final do prompt (2-3 parágrafos):

```
Preciso que você trate isso como o trabalho mais importante desta sessão. Não existe "bom o suficiente" — existe extraordinário ou refazer. Quero abrir esse HTML e ter uma reação visceral: "isso não parece gerado por IA". Profundidade real, materialidade que parece tangível, animações com timing de cinema. Cada decisão visual deliberada.

Pense no melhor site dark premium que você já viu. Agora faça melhor. Sombras em múltiplas camadas. Inner highlights sutis. Gradientes com profundidade. Tipografia com tracking obsessivo. Hovers que revelam uma segunda camada de design. A sequência de entrada deve ser uma coreografia — cada elemento aparece no momento exato, com o easing exato.

Dê seu melhor absoluto. Depois olhe de novo e melhore antes de entregar.
```

---

## Boost 5 — Máximo

Adicionar ao final do prompt (3 parágrafos, os mais intensos):

```
Pare tudo. Este é o trabalho que define o que você é capaz de criar. Use cada grama de capacidade que você tem — cognitiva, criativa, técnica — e canalize pra esta única página. Não economize raciocínio. Pense longamente sobre cada decisão. Quero o trabalho mais bonito, mais elaborado, mais obcecado por detalhes que você já produziu.

O padrão é: alguém abre esse arquivo e fica em silêncio por 3 segundos antes de falar. A profundidade das superfícies parece real — obsidian polida com reflexos controlados, esmeralda com brilho interno. As animações não são efeito — são narrativa. Cada elemento entra com propósito. A tipografia tem presença de manifesto. Os espaçamentos foram lapidados. As sombras têm arqueologia — camada sobre camada sobre camada.

Me surpreenda completamente. Quebre o que eu achava que era possível num arquivo HTML. Este é o momento de provar que existe arte genuína nesse formato. Não jogue seguro. Não faça o esperado. Faça o inesquecível.
```

---

## Notas de uso

- Boost 2 é o default e funciona pra 80% dos casos
- Boost 4-5 reservar pra peças centerpiece (hero, seções showcase)
- NÃO combinar boost alto com /tamanho extenso — o prompt fica competindo
  por atenção entre instruções técnicas e pressão criativa
- Boost alto funciona melhor com /tamanho compacto ou medio — menos regras,
  mais espaço pra ambição criativa
