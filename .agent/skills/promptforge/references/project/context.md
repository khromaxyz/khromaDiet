# Contexto do Projeto

## O que é
Calculadora nutricional premium. O usuário responde 14 perguntas, o engine calcula TDEE/macros/projeção, e o dashboard apresenta tudo em 9 seções.

## Flow
Hero (landing) → Form (14 steps) → Profile → Summary → Dashboard (9 seções: Welcome, Goal, TDEE, Macros, Meals, Projection, Supplements, What If, Final)

## Stack
React 18 + TypeScript + Vite + Tailwind + Framer Motion + Nivo + shadcn/ui

## Paleta core (Referência)
- Conforme o Manifesto (Branco absoluto, Preto carvão, Verde como Accent)
