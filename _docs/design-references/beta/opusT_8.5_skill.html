Foundation 06

Motion

Life, not spectacle. Fast, dry hovers. Smooth entrances with considered stagger. Spring easing for playful micro-interactions. Every animation reinforces hierarchy and quality.

Easing Curves

ease-out cubic-bezier(0.16, 1, 0.3, 1)
Primary. Most transitions.
ease-spring cubic-bezier(0.34, 1.56, 0.64, 1)
Toggles, switches, playful micro-interactions.
ease-in-out cubic-bezier(0.45, 0, 0.55, 1)
Background transitions, mode changes.
linear linear
Spinners, progress, continuous loops.

Duration Scale

--dur-fast 130ms Hover, focus, active

--dur-slow 360ms Panels, modals, reveals
--dur-slower 500ms Complex animations, charts
