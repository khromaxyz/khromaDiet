01 — Foundation

Design Principles

Five non-negotiable beliefs that shape every decision. If a component, animation or layout choice cannot trace back to one of these principles, it does not belong in the system.

01

Structural Clarity

Every element earns its space. No ornament without function. Borders, blocks and surfaces communicate hierarchy — not decoration.

02

Clinical Confidence

Generous whitespace, precise alignment and restrained palette create the feeling of a system that knows exactly what it's doing.

03

Tactile Precision

Interactive elements feel solid and responsive. Weight, depth and motion feedback make digital surfaces feel physically grounded.

04

Purposeful Motion

Animation exists only to orient, confirm or reveal. Every transition has a reason. Zero decorative movement. Zero delays without meaning.

05

Essential Completeness

Ship less, but ship finished. Every state is designed: empty, loading, error, success, edge. Nothing is left as an afterthought.

02 — Chromatic

Color System

Three colors. No exceptions. White for surface. Black for structure. Green for energy. All sophistication comes from contrast, transparency and material — never from palette expansion.

08 — Accessibility

A11Y Standards

WCAG AA compliance across all interactive elements. Keyboard navigation, focus management, semantic HTML and proper ARIA attributes are not optional — they are structural.

A11Y

Contrast Ratios

Black on White: 21:1. Green (#00E46A) on Black: 8.2:1. Green on White: 2.6:1 (decorative only, never for readable text). All body text meets AA minimum of 4.5:1.

A11Y

Keyboard Navigation

All interactive elements are focusable via Tab. Visible focus rings use green outline with 2px offset. Escape closes modals. Enter activates buttons. Space toggles controls.

A11Y

Semantic Structure

Proper heading hierarchy (h1-h6). ARIA labels on icon-only buttons. Role attributes on custom controls. Alt text policy. Landmark regions for screen readers.
