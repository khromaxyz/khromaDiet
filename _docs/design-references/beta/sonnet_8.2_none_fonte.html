Este documento governa toda a linguagem visual do produto. Cada token, componente e padrão aqui definido é a régua oficial para Hero, Form, Summary e Dashboard. Neo-Brutalismo Clean: força estrutural, contraste deliberado, acabamento de software maduro.

01

Princípios de Linguagem

Seis princípios não-negociáveis que regem toda decisão visual. Se uma escolha viola um desses princípios, ela está errada.

P.01
Força Estrutural

Bordas têm peso. Contornos têm intenção. Superfícies têm presença. Nada flutua sem razão.

P.02
Precisão Clínica

Números, métricas e rótulos respiram com exatidão. Cada dado é legível à primeira vista.

P.03
Silêncio Intencional

Espaço em branco é conteúdo. Margem é decisão. Densidade é sparse por escolha, não por descuido.

P.04
Disciplina Cromática

Um único acento. Verde vivo para energia, foco e ação. Sem deriva. Sem exceção decorativa.

P.05
Tato Humano

Brutalismo sem agressividade. Rigor sem frieza. O sistema é preciso e, ao mesmo tempo, acessível.

P.06
Motion como Hierarquia

Animação existe para orientar, confirmar e responder. Nunca para decorar ou impressionar.

Do / Don't — Uso de Cor
✓ Faça

Use verde exclusivamente em CTAs primários, estados de foco, valores positivos, confirmações e pontos de energia visual. Reserve-o para o que importa.

✗ Evite

Nunca use azul, roxo, dourado, coral ou outras famílias cromáticas. Criatividade aparece na composição, não na dispersão de cor.

Regras do Dark Mode
✓ Faça

Use #080808 como base. Preto abissal, sólido, imersivo. Superfícies sobem em passos de 10-13% de lightness.

✗ Evite

Nunca use cinzas médios preguiçosos como #333, #444 como base. Dilui a identidade abissal do dark mode.
